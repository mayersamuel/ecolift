# nuxt-starter

A simple nuxtStarter Project to get into coding more quickly.

## Installation

To install the dependencies, run:

```bash
  npm install
```

## Development
To start the development server, run:

```bash
  npm run dev
```

## Build
To build the project for production, run:

```bash
  npm run build
```

## Start
To start the production server, run:

```bash
  npm start
```
